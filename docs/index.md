# Початок

Це коротка навігаційна сторінка для власника магазину.

## З чого почати

Якщо ви відкрили документацію вперше, рухайтеся в такому порядку:

1. [Інструкція для власника](owner-guide.md) — основний гайд по роботі з магазином.
2. [Запуск магазину](launch.md) — короткий чекліст перед першим запуском.
3. [Оплати](payments.md) — як налаштувати оплати, зокрема Monobank Merchant.
4. [Нова пошта](nova-poshta.md) — як налаштувати доставку через Нову пошту.

## Яку сторінку відкривати в якій ситуації

- Якщо потрібен повний гайд по роботі з магазином: [Інструкція для власника](owner-guide.md)
- Якщо готуєте магазин до запуску: [Запуск магазину](launch.md)
- Якщо підключаєте оплату: [Оплати](payments.md)
- Якщо налаштовуєте доставку: [Нова пошта](nova-poshta.md)
