# Нова пошта

## Правильний порядок налаштування

Для отримання 'API ключа' вам потрібно:

1. Перейти на сторінку [`Нової пошти`](https://new.novaposhta.ua/).
2. Увійти в свій кабінет.
3. Налаштування - Безпека - `Створити Ключ`.
4. Скопіювати `API ключ`.

---
1. Відкрийте WebApp магазину.
2. Перейдіть у `Налаштування`.
3. Відкрийте секцію `Доставка`.
4. Знайдіть картку **Нова пошта**.
5. Вставте `API ключ` і натисніть `Зберегти API ключ`.
6. У полі `Місто відправника` знайдіть своє місто через `Знайти місто` і виберіть його зі списку.
7. У полі `Точка відправлення` знайдіть відділення або поштомат через `Знайти точку відправлення` і виберіть його зі списку.
8. Заповніть `ПІБ відправника`.
9. Заповніть `Телефон відправника`.
10. Натисніть `Зберегти доставку`.

---

## Що обов’язково має бути заповнено

Для коректної роботи потрібні:

- API ключ Нової пошти
- місто відправника
- точка відправлення
- ПІБ відправника
- телефон відправника

!!! warning "Не вводьте місто і відділення вручну навмання"
    У поточному інтерфейсі ці значення мають вибиратися зі списків пошуку, щоб збереглися коректні refs. Просто тексту в полі недостатньо.

---

## Коли ТТН створиться без помилки

Щоб ТТН створювалась коректно, мають бути одночасно виконані такі умови:

- увімкнений модуль Нової пошти
- збережений API ключ
- заповнені дані відправника
- в замовленні є дані отримувача
- в замовленні є Nova Poshta refs для міста і відділення отримувача
